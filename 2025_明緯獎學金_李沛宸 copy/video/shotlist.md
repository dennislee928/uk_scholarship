# 影片拍攝清單

## 總體規格
- **長度**：2–3 分鐘
- **題目**：「明緯獎學金能帶給自己什麼價值？」
- **必須回答四題**：優勢/劣勢與幫助、最大困難與解法、十年後、如何發揮價值

## 分段拍攝指南

### 第一段：優勢/劣勢與幫助（0:00–0:45，45 秒）
**量化重點**：強調具體能力與成果，說明獎學金如何放大價值

**必講量化點**：
- [ ] 優勢：安全開發流程落地能力（例：降低 X% 安全漏洞）
- [ ] 劣勢：海外進修成本高（具體金額/時間壓力）
- [ ] 獎學金幫助：節省 X 小時工作時間，專注學習產出

**拍攝重點**：
- 眼神自信，說明具體成果
- 語速平穩，數據說清楚
- 表情真誠，顯示對專業的熱情

### 第二段：最大困難與解法（0:45–1:20，35 秒）
**量化重點**：展示問題解決能力，強調方法與行動力

**必講量化點**：
- [ ] 困難：高強度專案下維持學習（具體時間分配）
- [ ] 解法：里程碑拆分法（每週產出 X 項成果）
- [ ] 驗證：實際案例（X 專案成功交付）

**拍攝重點**：
- 肢體語言：手勢輔助說明方法
- 節奏：先描述問題，再解釋解法
- 結論：強調持續改進的習慣

### 第三段：十年後想成為誰（1:20–2:00，40 秒）
**量化重點**：具體職涯目標與社會影響，連結 SDGs

**必講量化點**：
- [ ] 職涯定位：國際團隊安全標準領導者
- [ ] 具體成果：每年幫助 X 人提升安全能力
- [ ] SDGs 連結：優質教育（目標 4）與和平正義（目標 16）

**拍攝重點**：
- 視線向上，顯示遠大目標
- 聲音堅定，傳達承諾
- 表情充滿希望與決心

### 第四段：如何發揮獎學金價值（2:00–2:30，30 秒）
**量化重點**：明確可交付成果，回饋社會價值

**必講量化點**：
- [ ] 第一個成果：可部署安全專案範例（X 個下載）
- [ ] 第二個成果：開源安全工具包與教案（X 套）
- [ ] 第三個成果：工作坊分享（X 場，觸及 X 人）

**拍攝重點**：
- 結尾呼應開頭，形成完整價值循環
- 語氣積極，顯示回饋社會的熱情
- 眼神直接面對鏡頭，傳達真誠

## 拍攝前準備

### 環境設定
- [ ] 背景簡潔專業（淺色牆壁為佳）
- [ ] 燈光充足均勻（自然光或環形燈）
- [ ] 鏡頭高度與眼睛同高
- [ ] 收音清晰（使用外部麥克風）

### 個人準備
- [ ] 服裝：正式但不拘謹（襯衫或 polo 衫）
- [ ] 髮型：整潔自然
- [ ] 表情：自信微笑，真誠自然
- [ ] 語速：每分鐘 150–180 字

### 技術準備
- [ ] 手機/相機穩定固定
- [ ] 測試錄音品質
- [ ] 預錄一分鐘測試片段
- [ ] 準備計時器確保時間分配

## 後製檢查

### 內容檢查
- [ ] 四個題目全部回答
- [ ] 每個量化點都有具體數字
- [ ] 總長度在 2–3 分鐘
- [ ] 中文發音清晰流暢

### 技術檢查
- [ ] 畫面穩定無晃動
- [ ] 聲音清楚無雜音
- [ ] 光線均勻不過曝
- [ ] 檔案格式支援（MP4 建議）

### 最終檢查
- [ ] 檔案大小適中（建議 < 100MB）
- [ ] 檔名清晰標註（例：明緯獎學金申請影片_李沛宸.mp4）
- [ ] 準備雲端連結以便提交
