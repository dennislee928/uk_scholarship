# 官方規格驗收條件

## 申請期間
每年 5 月至 7/31（逾期不受理）。

## 寄送方式
完整文件於期限內寄到 info@meanwellfoundation.org。

## PDF 合併規則
文件 1–4 必須整合成單一 PDF，檔名為 2025明緯獎學金-XXX(姓名)。

## 影片規格
2–3 分鐘，題目是「明緯獎學金能帶給自己什麼價值？」並回答四題（優勢/劣勢與幫助、最大困難與解法、十年後、如何發揮價值）。

---

## 申請人資格檢查

- [ ] 確認具備中華民國國籍（台灣護照，戶籍登記在台灣）
- [ ] 確認當年度確定要出國留學（含碩士）
- [ ] 確認已取得錄取通知書或入學證明

## 5 項附件完整性檢查

- [ ] **申請書**：已填寫官方申請表單，含 300 字短答
- [ ] **自傳與學習計畫**：自傳（800 字內）、短期學習計畫（800 字內）、未來工作應用（1000 字內）
- [ ] **錄取通知或入學證明**：當年度錄取學校通知（UofG offer），有明確日期
- [ ] **能力證明文件**：推薦信、證照、作品集、獲獎證明、語言成績等，所有文件都有日期與出處
- [ ] **影片連結**：2–3 分鐘影片，已上傳至 YouTube 或 Google Drive，設定為可公開讀取

## 影片 4 題對應檢查

- [ ] **第一題**：優勢與劣勢；獎學金如何幫助（30–45 秒）
- [ ] **第二題**：過去最大困難 + 如何解決（30–45 秒）
- [ ] **第三題**：十年後想成為什麼樣的人（30–45 秒）
- [ ] **第四題**：如何發揮此獎學金價值（成果承諾）（30–45 秒）

## 最終驗收檢查表

- [ ] 確認申請時間在 5/1–7/31 之間
- [ ] 確認寄送到正確 email：info@meanwellfoundation.org
- [ ] 確認 PDF 為單一檔案，檔名為 `2025明緯獎學金-李沛宸.pdf`（無空格）
- [ ] 確認影片長度為 2–3 分鐘
- [ ] 確認影片回答全部四個指定題目
- [ ] 確認所有必要文件已包含在 PDF 中
- [ ] 確認所有附件都有日期與出處標註
- [ ] 確認已完成 [CHECKLIST.md](../../CHECKLIST.md) 中的所有檢查項目
