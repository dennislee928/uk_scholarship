## 二、PDF 合併流程

### 2.1 合併順序

請按照以下順序將文件合併為單一 PDF：

1. **申請書**（含 300 字短答）
2. **自傳與學習計畫**（自傳、短期學習計畫、未來工作應用）
3. **錄取通知/在學證明**
4. **能力證明附錄**（推薦信、證照、作品集等）
5. **附件對照表**（可選，建議放在最後）

### 2.2 檔名格式

**重要**：最終 PDF 檔名必須為：`2025明緯獎學金-李沛宸.pdf`（注意：無空格）

### 2.3 PDF 合併工具建議

#### macOS

**方法一：使用預覽程式**
1. 開啟第一個 PDF 文件
2. 在側邊欄中，拖拽其他 PDF 文件到第一個文件下方
3. 調整順序（拖拽即可）
4. 檔案 > 匯出為 PDF
5. 儲存為 `2025明緯獎學金-李沛宸.pdf`

**方法二：使用命令列**
```bash
# 安裝 poppler（如果尚未安裝）
brew install poppler

# 合併 PDF
pdfunite "01_申請書/300字短答_為何申請.pdf" \
         "02_自傳與學習計畫/自傳.pdf" \
         "02_自傳與學習計畫/短期學習計畫.pdf" \
         "02_自傳與學習計畫/未來工作應用.pdf" \
         "03_錄取在學證明/[錄取證明].pdf" \
         "04_能力證明附錄/[附件1].pdf" \
         "04_能力證明附錄/[附件2].pdf" \
         "2025明緯獎學金-李沛宸.pdf"
```

#### Windows

**方法一：使用 Adobe Acrobat**
1. 開啟 Adobe Acrobat
2. 工具 > 合併檔案
3. 選擇所有 PDF 文件
4. 調整順序
5. 合併檔案
6. 另存為 `2025明緯獎學金-李沛宸.pdf`

**方法二：使用線上工具**
1. 前往 https://www.ilovepdf.com/merge-pdf
2. 上傳所有 PDF 文件
3. 調整順序
4. 合併 PDF
5. 下載並命名為 `2025明緯獎學金-李沛宸.pdf`

#### Linux

**使用 PDFtk**
```bash
# 安裝 PDFtk
sudo apt-get install pdftk

# 合併 PDF
pdftk "01_申請書/300字短答_為何申請.pdf" \
      "02_自傳與學習計畫/自傳.pdf" \
      "02_自傳與學習計畫/短期學習計畫.pdf" \
      "02_自傳與學習計畫/未來工作應用.pdf" \
      "03_錄取在學證明/[錄取證明].pdf" \
      "04_能力證明附錄/*.pdf" \
      cat output "2025明緯獎學金-李沛宸.pdf"
```

## 三、影片連結處理

### 3.1 上傳影片

**選項一：YouTube**
1. 上傳影片到 YouTube
2. 設定為「公開」或「不公開但知道連結的人可觀看」
3. 複製影片連結

**選項二：Google Drive**
1. 上傳影片到 Google Drive
2. 右鍵 > 共用 > 設定為「知道連結的使用者」
3. 複製共用連結

### 3.2 在 PDF 中加入影片連結

建議在 PDF 第一頁加入影片連結，格式如下：

```
影片連結：[在此貼上 YouTube 或 Google Drive 連結]
```

或建立一個單獨的「申請摘要頁」，包含：
- 申請人姓名
- 就讀學校與科系
- 影片連結
- 申請日期

## 七、常見問題

### Q: PDF 檔名可以有空格嗎？
A: **不可以**。檔名必須為 `2025明緯獎學金-李沛宸.pdf`（無空格）。

### Q: 影片可以附在 Email 中嗎？
A: **不可以**。影片必須上傳到 YouTube 或 Google Drive，並在 Email 中提供連結。

### Q: 可以分多次寄送文件嗎？
A: **不建議**。所有文件應合併為單一 PDF 一次寄送。

### Q: 如果超過申請期間怎麼辦？
A: 逾期不受理，請等待下一年度的申請期間。
