# 300 字短答：為何申請明緯獎學金

我申請明緯獎學金，是為了完成 2026/09 於英國格拉斯哥大學 Software Development MSc 的進修計畫，並把所學落地回饋社會。

**現有專長與實績**：我目前在資安與全端工程領域工作，在統一集團專案中以 JavaScript + Terraform + Python 完成 4000 個 DNS 與 WAF 配置自動化（20 分鐘完成，人工需 5 工作天）。過去 3 年建立 5 套自動化安全檢測腳本，降低資安事故率 30%，並開發 3 個開源 DevSecOps 工具，累積超過 200 次下載。

**獎學金用途**：此 50 萬獎學金將優先用於學費（約 35 萬）與必要生活支出（約 15 萬），降低我在就學期間的財務壓力，使我能專注於課業與可交付成果。

**回饋承諾與 SDGs 對齊**：我規劃的「社會安全與數位韌性」計畫分三層：技術研發（去識別化案例庫 ≥ 500 份、開源安全分析工具）、跨域知識設計（至少 3 套跨域教案）、大眾教育服務（至少 12 場工作坊、至少 300 名參與者、安全素養提升 ≥ 25%）。這些成果對齊 SDG4、SDG8 與 SDG16，透過可衡量指標回饋社會。
