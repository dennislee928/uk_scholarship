# 2025 明緯獎學金申請包

## 資料夾結構說明

本申請包包含以下資料夾：

### 01\_申請書/

包含 300 字短答初稿，可直接用於明緯獎學金申請表單。

### 02\_自傳與學習計畫/

包含三個文件：

- **自傳.md**：自傳提綱（800 字內）
- **短期學習計畫.md**：學習計畫提綱（800 字內）
- **未來工作應用.md**：未來工作應用說明（1000 字內）

### 03\_錄取在學證明/

放置錄取通知書、在學證明等相關文件。

### 04\_能力證明附錄/

放置學歷證明、證照、專案作品、獲獎證明等能力證明文件。

### 05\_影片/

包含影片逐字稿骨架，用於錄製 2-3 分鐘的申請影片。

## 最終 PDF 檔名

根據明緯獎學金要求，最終合併的 PDF 檔名應為：
**2025 明緯獎學金-李沛宸.pdf**

詳見 `檔名規範.md`。

## 使用流程

1. **填寫內容**：根據各文件的提綱結構，填入個人資訊與具體內容
2. **擴寫潤飾**：將提綱擴寫成完整文章，注意字數限制
3. **準備證明文件**：將相關證明文件放入對應資料夾
4. **錄製影片**：根據逐字稿骨架錄製申請影片
5. **合併 PDF**：將所有文件合併成最終 PDF，檔名為 `2025 明緯獎學金-李沛宸.pdf`
