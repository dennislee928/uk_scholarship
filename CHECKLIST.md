# 明緯獎學金送件前檢查清單

## 重要提醒
**送件前請逐項檢查，確保所有項目都已完成，任何不合規都可能導致申請被直接退回或不受理。**

---

## 一、申請資格檢查

- [ ] **中華民國國籍確認**：具備台灣護照，戶籍登記在台灣
- [ ] **當年度出國留學確認**：確定於 2025 或 2026 年出國留學（含碩士）
- [ ] **申請期間確認**：送件時間在 **每年 5 月 1 日至 7 月 31 日**之間（逾期不受理）

---

## 二、申請文件完整性檢查（5 項必備）

### 2.1 申請書
- [ ] 已填寫明緯官方申請表單
- [ ] **300 字短答已完成**，包含：
  - [ ] 就讀學校與入學時間
  - [ ] 現有專長與實績（含量化數據）
  - [ ] 50 萬獎學金用途（學費/生活費缺口）
  - [ ] 回饋承諾（對齊 SDGs，含具體 KPI）

### 2.2 自傳與學習計畫
- [ ] **自傳**已完成（800 字內）
- [ ] **短期學習計畫**已完成（800 字內），包含：
  - [ ] 時間線表格（Month 1-12）
  - [ ] 每階段目標與交付物
- [ ] **未來工作應用**已完成（1000 字內），包含：
  - [ ] SDGs 對照表
  - [ ] 國際貢獻與社群影響說明

### 2.3 錄取通知或入學證明
- [ ] 已取得 University of Glasgow 正式錄取通知書
- [ ] 錄取通知書有明確日期
- [ ] 若錄取獎學金資格，已附上相關證明

### 2.4 能力證明文件
- [ ] **推薦信**：至少 1 封，有日期與推薦人簽名
- [ ] **語言成績證明**：IELTS/TOEFL 等，有日期與機構名稱
- [ ] **專業證照**：資安/DevOps 相關證照（如有）
- [ ] **作品集**：專案作品、開源貢獻證明（如有）
- [ ] **獲獎證明**：相關獎項或成就證明（如有）
- [ ] 所有證明文件都有**日期與出處標註**

### 2.5 影片連結
- [ ] 影片已錄製完成（2-3 分鐘）
- [ ] 影片已上傳至 YouTube 或 Google Drive
- [ ] 影片設定為**可公開讀取**
- [ ] 影片連結已準備好，可放在 PDF 第一頁摘要
- [ ] 影片內容回答官方 4 題：
  - [ ] 優勢與劣勢；獎學金如何幫助
  - [ ] 過去最大困難 + 如何解決
  - [ ] 十年後想成為什麼樣的人
  - [ ] 如何發揮此獎學金價值（成果承諾）

---

## 三、PDF 合併與檔名檢查

- [ ] **所有文件（1-4 項）已合併為單一 PDF**
- [ ] **PDF 檔名格式正確**：`2025明緯獎學金-李沛宸.pdf`（注意：無空格）
- [ ] PDF 合併順序正確：
  1. 申請書（含 300 字短答）
  2. 自傳與學習計畫
  3. 錄取通知/在學證明
  4. 能力證明附錄
- [ ] PDF 第一頁包含影片連結（如適用）
- [ ] PDF 檔案大小適中（建議 < 10MB）
- [ ] PDF 所有頁面清晰可讀

---

## 四、內容品質檢查

### 4.1 量化數據檢查
- [ ] 所有專案成果都有具體數字（例如：年減資安事故率 X%、建立 N 套工具）
- [ ] 回饋承諾都有具體 KPI（例如：X 篇教學、Y 場分享會、Z 個開源工具）
- [ ] 時間線表格中的交付物都是可驗證的

### 4.2 SDGs 對齊檢查
- [ ] 明確標示對應的 SDG 目標（SDG4、SDG8、SDG9、SDG16 等）
- [ ] 說明如何透過具體行動達成這些目標
- [ ] SDGs 對照表已包含在相關文件中

### 4.3 語言與格式檢查
- [ ] 所有文件使用繁體中文
- [ ] 無錯別字與語法錯誤
- [ ] 格式一致、專業
- [ ] 建議請他人協助檢查錯字與語句流暢度

---

## 五、附件命名與歸檔檢查

- [ ] 所有附件遵循統一命名規則：
  - 推薦信：`2025_Meanwell_RecoLetter_Prof_X.pdf`
  - 語言證明：`2025_Meanwell_Cert_IELTS.pdf`
  - 作品集：`2025_Meanwell_Portfolio_ProjectA.pdf`
  - 證照：`2025_Meanwell_Cert_[證照名稱].pdf`
- [ ] 附件對照表已建立，列出所有附件及其出處
- [ ] 所有附件都有日期標註

---

## 六、送件準備檢查

### 6.1 Email 準備
- [ ] Email 主旨格式：`2025明緯獎學金申請 – 李沛宸 – MSc Software Development – University of Glasgow`
- [ ] Email 內文包含：
  - [ ] PDF 附件說明
  - [ ] 影片連結
  - [ ] 簡要申請說明
- [ ] 收件地址確認：`info@meanwellfoundation.org`

### 6.2 最終確認
- [ ] 已閱讀 `requirements.md` 中的所有官方規格
- [ ] 已對照 `送件指南.md` 完成所有步驟
- [ ] 已備份所有原始文件
- [ ] 已確認送件時間在申請期間內（5/1-7/31）
