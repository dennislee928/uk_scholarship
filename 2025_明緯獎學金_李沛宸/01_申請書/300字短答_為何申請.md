# 300 字短答：為何申請明緯獎學金

我申請明緯獎學金，是為了完成 2026/09 於英國格拉斯哥大學 Software Development MSc 的進修計畫，並把所學落地回饋社會。

**現有專長與實績**：我目前在資安與全端工程領域工作，有效整合工程、自動化與雲端技術。在統一集團專案中，我以 JavaScript + Terraform + Python 完成 4000 個 DNS 與 WAF 配置自動化（20 分鐘完成，人工預計需 5 個工作天），顯著提升部署效率與風險控制。此外，過去 3 年建立 5 套自動化安全檢測腳本，協助團隊降低資安事故率 30%，並開發 3 個開源 DevSecOps 工具，累積超過 200 次下載。

**獎學金用途**：此 50 萬獎學金將優先用於學費（約 35 萬）與必要生活支出（約 15 萬），降低我在就學期間的財務壓力，使我能專注於課業與可交付成果，無需分心於打工。

**回饋承諾與 SDGs 對齊**：我規劃的「社會安全與數位韌性」計畫分三個層次：技術研發（建立去識別化案例庫 ≥ 500 份樣本、開源安全分析工具）、跨域知識設計（完成至少 3 套跨域教案、建立知識圖譜）、面向大眾的教育服務（舉辦至少 12 場工作坊、教育至少 300 名參與者、安全素養提升 ≥ 25%）。這些成果對齊 **SDG4（優質教育）**、**SDG8（體面工作與經濟成長）** 與 **SDG16（和平、正義與健全制度）**，透過可衡量的指標回饋社會，促進數位安全素養與跨社群教育資源的永續提升。
