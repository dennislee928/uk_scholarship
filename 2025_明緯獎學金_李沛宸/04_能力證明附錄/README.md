# 04_能力證明附錄

## 需要準備的文件

此資料夾應包含以下能力證明文件：

### 1. 學歷證明
- **大學畢業證書**：掃描檔或 PDF
- **大學成績單**：官方成績單掃描檔
- **檔名建議**：`大學畢業證書.pdf`、`大學成績單.pdf`

### 2. 專業證照
- **資安相關證照**：[列出具體證照名稱，例如：CISSP、CEH、Security+ 等]
- **DevOps/雲端證照**：[列出具體證照名稱，例如：AWS Certified、Kubernetes Certified 等]
- **其他技術證照**：[列出其他相關證照]
- **檔名建議**：`證照_[證照名稱].pdf`

### 3. 專案作品集
- **開源專案證明**：GitHub 專案連結清單或截圖
- **專案成果展示**：專案說明文件、Demo 影片連結等
- **檔名建議**：`專案作品集.md` 或 `專案_[專案名稱].pdf`

### 4. 獲獎證明
- **競賽獲獎**：技術競賽、黑客松等獲獎證明
- **學術獎項**：學術成就、獎學金等證明
- **檔名建議**：`獲獎證明_[獎項名稱].pdf`

### 5. 技術文章/發表
- **技術文章**：發表過的技術文章清單或 PDF 檔
- **技術分享**：演講、工作坊等活動證明
- **檔名建議**：`技術文章清單.md` 或 `發表_[文章名稱].pdf`

### 6. 推薦信（如有）
- **推薦信**：來自導師、主管或業界專家的推薦信
- **檔名建議**：`推薦信_[推薦人姓名].pdf`

### 7. 其他能力證明
- **語言能力證明**：英文檢定成績單（IELTS、TOEFL 等）
- **實習證明**：相關工作或實習經驗證明
- **檔名建議**：`語言能力證明.pdf`、`實習證明.pdf`

## 文件組織建議

建議建立一個索引文件 `能力證明索引.md`，列出所有文件及其說明：

```markdown
# 能力證明索引

## 學歷證明
1. 大學畢業證書.pdf
2. 大學成績單.pdf

## 專業證照
1. 證照_[證照名稱].pdf
...

## 專案作品
1. [專案名稱] - GitHub: [連結]
...

## 獲獎證明
1. 獲獎證明_[獎項名稱].pdf
...
```

## 注意事項

- 所有文件需為清晰可讀的 PDF 或高解析度掃描檔
- 建議按照類別組織文件，便於查找
- 保留原始文件，並準備多份備份
- 如文件為英文，建議在文件名稱中標註
- 確保所有證明文件都在有效期限內
- 建議準備一份簡要的能力證明摘要，方便審查委員快速了解
