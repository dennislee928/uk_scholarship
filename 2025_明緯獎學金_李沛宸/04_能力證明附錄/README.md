# 04_能力證明附錄

## 需要準備的文件

此資料夾應包含以下能力證明文件：

### 1. 學歷證明
- **大學畢業證書**：掃描檔或 PDF
- **大學成績單**：官方成績單掃描檔
- **統一命名規則**：
  - `2025_Meanwell_Diploma_Bachelor.pdf`
  - `2025_Meanwell_Transcript_Bachelor.pdf`

### 2. 專業證照
- **資安相關證照**：[列出具體證照名稱，例如：CISSP、CEH、Security+ 等]
- **DevOps/雲端證照**：[列出具體證照名稱，例如：AWS Certified、Kubernetes Certified 等]
- **其他技術證照**：[列出其他相關證照]
- **統一命名規則**：`2025_Meanwell_Cert_[證照名稱].pdf`
  - 範例：`2025_Meanwell_Cert_CISSP.pdf`
  - 範例：`2025_Meanwell_Cert_AWS_Solutions_Architect.pdf`

### 3. 專案作品集
- **開源專案證明**：GitHub 專案連結清單或截圖
- **專案成果展示**：專案說明文件、Demo 影片連結等
- **統一命名規則**：`2025_Meanwell_Portfolio_[專案名稱].pdf`
  - 範例：`2025_Meanwell_Portfolio_DevSecOps_Toolkit.pdf`
  - 範例：`2025_Meanwell_Portfolio_Secure_By_Design_Project.pdf`

### 4. 獲獎證明
- **競賽獲獎**：技術競賽、黑客松等獲獎證明
- **學術獎項**：學術成就、獎學金等證明
- **統一命名規則**：`2025_Meanwell_Award_[獎項名稱].pdf`
  - 範例：`2025_Meanwell_Award_Hackathon_2024.pdf`

### 5. 技術文章/發表
- **技術文章**：發表過的技術文章清單或 PDF 檔
- **技術分享**：演講、工作坊等活動證明
- **統一命名規則**：
  - 文章清單：`2025_Meanwell_Publications_List.pdf`
  - 單篇文章：`2025_Meanwell_Publication_[文章名稱].pdf`
  - 演講證明：`2025_Meanwell_Talk_[活動名稱].pdf`

### 6. 推薦信（如有）
- **推薦信**：來自導師、主管或業界專家的推薦信
- **統一命名規則**：`2025_Meanwell_RecoLetter_[推薦人姓名或職稱].pdf`
  - 範例：`2025_Meanwell_RecoLetter_Prof_Wang.pdf`
  - 範例：`2025_Meanwell_RecoLetter_Manager_Lee.pdf`

### 7. 其他能力證明
- **語言能力證明**：英文檢定成績單（IELTS、TOEFL 等）
- **實習證明**：相關工作或實習經驗證明
- **統一命名規則**：
  - 語言證明：`2025_Meanwell_Cert_IELTS.pdf` 或 `2025_Meanwell_Cert_TOEFL.pdf`
  - 實習證明：`2025_Meanwell_Internship_[公司名稱].pdf`

## 附件對照表模板

在合併 PDF 時，請建立附件對照表，列出所有附件及其出處。以下為模板：

| 附件編號 | 附件名稱 | 附件類型 | 日期 | 出處/來源 |
|---------|---------|---------|------|----------|
| 附件1 | 2025_Meanwell_RecoLetter_Prof_Wang.pdf | 推薦信 | 2025-XX-XX | XX大學XX教授 |
| 附件2 | 2025_Meanwell_Cert_IELTS.pdf | 語言證明 | 2025-XX-XX | British Council |
| 附件3 | 2025_Meanwell_Portfolio_DevSecOps_Toolkit.pdf | 作品集 | 2024-XX-XX | GitHub: [連結] |
| 附件4 | 2025_Meanwell_Cert_CISSP.pdf | 專業證照 | 2024-XX-XX | (ISC)² |
| 附件5 | 2025_Meanwell_Transcript_Bachelor.pdf | 成績單 | 202X-XX-XX | XX大學教務處 |

**注意事項**：
- 每個附件都必須有明確的日期標註
- 每個附件都必須標註出處或來源機構
- 附件對照表應放在合併 PDF 的最後一頁或第一頁摘要中

## 注意事項

- 所有文件需為清晰可讀的 PDF 或高解析度掃描檔
- 建議按照類別組織文件，便於查找
- 保留原始文件，並準備多份備份
- 如文件為英文，建議在文件名稱中標註
- 確保所有證明文件都在有效期限內
- 建議準備一份簡要的能力證明摘要，方便審查委員快速了解
