# 03_錄取在學證明

## 需要準備的文件

此資料夾應包含以下證明文件：

### 1. 錄取證明
- **文件類型**：PDF 或掃描檔
- **內容**：格拉斯哥大學（University of Glasgow）的正式錄取通知書
- **檔名建議**：`錄取通知書_University_of_Glasgow.pdf`

### 2. 在學證明（如已入學）
- **文件類型**：PDF 或掃描檔
- **內容**：學校出具的正式在學證明
- **檔名建議**：`在學證明_University_of_Glasgow.pdf`

### 3. 其他相關證明
- 如有其他相關證明文件（例如：獎學金通知、學費繳費證明等），也可放置於此資料夾

## 注意事項

- 所有文件需為清晰可讀的 PDF 或高解析度掃描檔
- 建議保留原始文件，並準備多份備份
- 如文件為英文，建議在文件名稱中標註
- 確保文件日期在有效期限內
